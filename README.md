# Coffee Shop Test

This repository is a small Astro-based starter site originally built as a generic "dev kit". It has been customized to represent an independent coffee shop called **Mocha Stop**.

The purpose of this commit is to **prove the developer's starter kit works** by:

- swapping out placeholder content with coffee-themed text and images
- applying a warm color palette and injecting a serif heading font
- keeping the original workflow (Tailwind, Astro components) intact
- running the development server (`npm run dev`) without errors

## Running the project

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser to see the live site. The pages include a hero section, feature blurb, sample menu items, testimonials, FAQ, and footer.

## What's been changed from the kit

- `tailwind.config.mjs` extended with a coffee palette and serif font
- Layout includes Google Fonts link for `Merriweather`
- Components updated with coffee‑shop copy and styling tweaks
- `src/pages/index.astro` populated with themed sections and sample data
- New README with instructions and demonstration notes

This project serves as a **portfolio piece** to show clients how you can adapt a template into a branded site. For your next project, you can focus on custom design elements or client-specific features, knowing the starter kit handles the plumbing.
